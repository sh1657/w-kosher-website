# 📁 תיקיית תוכן לפי שפות - Content by Language

## 🎯 מטרה

תיקייה זו מכילה את כל התוכן הטקסטואלי של האתר מחולק לפי שפות.
כל קובץ מכיל **רק טקסט נקי** ללא קוד HTML, כך שתוכל לערוך ולהגיה בקלות.

## 🔄 תהליך העבודה

### שלב 1: עריכה והגהה
1. פתח את התיקייה של השפה הרצויה
2. ערוך כל קובץ `.txt` - תקן שגיאות, שפר ניסוחים
3. שמור את השינויים

### שלב 2: יישום השינויים באתר
לאחר שסיימת לערוך, תן לי לדעת ואני:
1. אקרא את הקבצים המעודכנים
2. אטמיע את הטקסט המתוקן בקבצי ה-HTML
3. אשמור על כל העיצוב והפונקציונליות

## 📋 מפתח הדפים

| שם קובץ | דף באתר | מה כולל |
|---------|---------|---------|
| `homepage.txt` | דף הבית | כותרות, סיסמאות, סטטיסטיקות, יתרונות |
| `certification.txt` | דף הסמכה | תהליך ההסמכה, שלבים, דרישות |
| `companies.txt` | דף חברות | רשימת חברות מוסמכות, תעודות |
| `about-us.txt` | אודותינו | מי אנחנו, הצוות, הרבנים |
| `contact.txt` | צור קשר | פרטי ת יצירת קשר, טופס |
| `request-certification.txt` | בקשת הסמכה | טופס בקשה, שדות, הוראות |
| `privacy-policy.txt` | מדיניות פרטיות | תנאים משפטיים |
| `terms-of-use.txt` | תנאי שימוש | תנאים משפטיים |

## ✅ טיפים לעריכה

### כותרות
- כותרות ראשיות מסומנות ב-`===`
- כותרות משניות מסומנות ב-`---`
- כותרות קטנות מסומנות ב-`###`

### רשימות
- רשימות מסומנות עם `-` או `•`
- רשימות ממוספרות: `1.`, `2.`, `3.`

### הדגשות
- טקסט מודגש: `**טקסט**`
- ציטוטים: `"טקסט"`

### אימוג'ים
- אפשר להשאיר או להוסיף אימוג'ים
- הם מוסיפים חיות לטקסט
