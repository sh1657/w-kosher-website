# רשימת בדיקות לפני הפעלת המערכת

## ✅ בדיקות CSS

### בדיקה 1: העתקת CSS מלא
- [ ] העתקתי את כל התוכן מקובץ `wordpress-ready-css.css`
- [ ] הדבקתי ב-WordPress.com תחת Additional CSS
- [ ] לחצתי על Publish ושמרתי

### בדיקה 2: עיצוב כפתורי השפה
- [ ] הכפתורים מופיעים נכון בעמוד
- [ ] הכפתורים נראים יפים ובולטים
- [ ] יש hover effect כשעוברים עליהם עם העכבר

## ✅ בדיקות JavaScript

### בדיקה 3: הוספת קוד JavaScript
- [ ] העתקתי את כל התוכן מקובץ `wordpress-ready-js.js`
- [ ] הוספתי דרך Plugin או HTML Block
- [ ] הקוד נטען בכל הדפים

### בדיקה 4: פונקציונליות בסיסית
- [ ] לחיצה על כפתור משנה את השפה
- [ ] הכפתור הנלחץ הופך לצבע כחול (active)
- [ ] אין שגיאות בקונסול הדפדפן (F12)

## ✅ בדיקות תוכן

### בדיקה 5: עדכון תוכן דף הבית
- [ ] החלפתי טקסטים עם המבנה הרב-לשוני
- [ ] כל טקסט עטוף ב-span עם המחלקה הנכונה
- [ ] יש תוכן בשלוש השפות: `.text-pt`, `.text-en`, `.text-es`

### בדיקה 6: עדכון דף חברות
- [ ] הוספתי `class="secondary-page"` לדף
- [ ] עדכנתי את התוכן לפורמט רב-לשוני
- [ ] הדף נראה טוב בדסקטופ (לא כמו מובייל)

## ✅ בדיקות פונקציונליות

### בדיקה 7: שמירת השפה
- [ ] בוחר שפה כלשהי
- [ ] מרענן את הדף (F5)
- [ ] השפה שנבחרה נשארת אחרי הרענון

### בדיקה 8: מעבר בין דפים
- [ ] בוחר שפה בדף הבית
- [ ] עובר לדף אחר
- [ ] השפה שנבחרה נשמרת בדף החדש

## ✅ בדיקות רספונסיביות

### בדיקה 9: תצוגה במובייל
- [ ] פותח באייפון/אנדרואיד
- [ ] כפתורי השפה נראים טוב
- [ ] התוכן נקרא ונראה טוב
- [ ] המערכת פועלת בטלפון

### בדיקה 10: תצוגה בדסקטופ
- [ ] פותח במחשב/מחשב נייד
- [ ] דפים משניים נראים רחבים (לא צרים כמו מובייל)
- [ ] כל האלמנטים במקום הנכון

## ✅ בדיקות דפדפנים

### בדיקה 11: Chrome/Safari
- [ ] כל הפונקציות עובדות ב-Chrome
- [ ] כל הפונקציות עובדות ב-Safari

### בדיקה 12: Firefox/Edge (אם יש גישה)
- [ ] מערכת השפות פועלת
- [ ] אין שגיאות או בעיות עיצוב

## 🚨 פתרון בעיות

### אם כפתורי השפה לא עובדים:
1. לך ל-F12 → Console
2. חפש שגיאות אדומות
3. ודא שה-JavaScript נטען
4. בדוק שהכפתורים כוללים `onclick="changeLanguage('pt')"`

### אם השפה לא נשמרת:
1. נסה לנקות Cache של הדפדפן
2. ודא שה-localStorage מאופשר
3. בדוק שאין mode פרטי בדפדפן

### אם העיצוב נשבר:
1. ודא שה-CSS הועתק במלואו
2. חפש התנגשויות עם התמה
3. נסה להוסיף `!important` לכמה חוקי CSS
