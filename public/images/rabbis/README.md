- `default-rabbi.jpg` - תמונת ברירת מחדל במקרה של שגיאה

## דרישות טכניות:
- **גודל:** 400x400 פיקסלים מינימום
- **פורמט:** JPG או PNG
- **יחס גובה-רוחב:** 1:1 (ריבוע) או 4:3
- **איכות:** גבוהה, מתאימה לאתר מקצועי
- **רזולוציה:** 72-150 DPI
- **גודל קובץ:** עד 500KB לכל תמונה

## הערות:
- התמונות צריכות להיות מקצועיות ובאיכות גבוהה
- רצוי עם רקע נקי או מטושטש
- התמונות יוצגו בגודל 280px גובה בדף הבית
- במקרה של שגיאה בטעינה, תוצג התמונה default-rabbi.jpg
- השמות בדיוק כמו ברשימה - זה חשוב לתצוגה נכונה!
