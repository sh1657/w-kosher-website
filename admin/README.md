## 💾 ניהול תוכן

### עריכת טקסט פשוט

1. מצא את השדה שברצונך לערוך
2. לחץ בתוך השדה
3. ערוך את הטקסט
4. המערכת תזהה אוטומטית את השינויים

### עריכת רשימות (Arrays)

לדוגמה: תחומי התמחות של רבנים

```
מנהיגות ארגונית, הכשרה בינלאומית, תקני כשרות
```

**חשוב:** הפרד בין הפריטים עם פסיק (,) ורווח.

### שמירת שינויים

לחץ על הכפתור **💾 שמור שינויים** בראש העמוד.

אחרי לחיצה:
1. תקבל הודעת אישור "✅ השינויים נשמרו בהצלחה!"
2. המערכת תוריד אוטומטית קובץ JSON מעודכן למחשב שלך
3. העתק את הקובץ לתיקיית `admin/data/`

## 🔄 שמירה ופרסום

### שמירה מקומית

המערכת שומרת את השינויים ב-localStorage של הדפדפן.  
השינויים נשמרים גם כשאתה עובר בין שפות.

### שמירה קבועה

כאשר אתה לוחץ על "שמור שינויים":

1. **הורדת קובץ JSON** - הקובץ המעודכן יורד אוטומטית
2. **העתקה ידנית** - העתק את הקובץ לתיקיית `admin/data/`
3. **פרסום לאתר** - העלה את הקובץ לשרת (במידת הצורך)

### שמירה אוטומטית

המערכת שומרת אוטומטית כל 5 דקות ל-localStorage.  
זה מונע איבוד נתונים במקרה של בעיה טכנית.

## 🛠️ טיפים לשימוש

### 1. שמור לעיתים קרובות
לחץ על "שמור שינויים" כל כמה דקות כדי למנוע איבוד נתונים.

### 2. בדוק את השינויים
לפני שמירה סופית, עבור על כל השדות ובדוק שהטקסט נכון.

### 3. שמור גיבוי
שמור עותק של קבצי ה-JSON המקוריים לפני עריכות גדולות.

### 4. עבוד בשפה אחת בכל פעם
השלם את עריכת שפה אחת, שמור, ואז עבור לשפה הבאה.

### 5. השתמש בעריכה ידידותית למובייל
המערכת מותאמת לעבודה גם מהסמארטפון!

## ❓ שאלות נפוצות

### האם אני צריך להיות מחובר לאינטרנט?

לא בהכרח. המערכת עובדת מקומית על המחשב שלך.  
אבל תצטרך אינטרנט כדי להעלות את השינויים לשרת (בעתיד).

### מה קורה אם אני סוגר את הדפדפן בטעות?

אם שמרת את השינויים ב-localStorage, הם לא יאבדו.  
פתח שוב את העורך והשינויים עדיין יהיו שם.

### איך אני מפרסם את השינויים באתר?

כרגע, אתה מוריד את קובץ ה-JSON ומעתיק אותו ידנית.  
בעתיד ניתן להוסיף פרסום אוטומטי לשרת.

### האם אפשר להוסיף עוד שפות?

כן! פשוט:
1. צור קובץ JSON חדש (למשל `content-es.json` לספרדית)
2. הוסף טאב חדש ב-`editor.html`
3. עדכן את `editor.js` לטעון את השפה החדשה
