# תמונות הסמכות

תיקייה זו מיועדת ללוגואים של הסמכות ובתי הדין השונים.

## לוגואים נדרשים:

### הסמכות רבניות:
- `beth-din-jerusalem.png` - לוגו בית דין ירושלים
- `beth-din-london.png` - לוגו בית דין לונדון  
- `ou-kosher.png` - לוגו OU Kosher
- `star-k.png` - לוגו Star-K
- `default-cert.png` - לוגו ברירת מחדל במקרה של שגיאה

## דרישות טכניות:
- **גודל:** 120x120 פיקסלים מינימום
- **פורמט:** PNG מועדף (עם רקע שקוף)
- **יחס גובה-רוחב:** 1:1 (ריבוע) מועדף
- **איכות:** גבוהה וחדה
- **רזולוציה:** 72-150 DPI
- **גודל קובץ:** עד 200KB לכל לוגו

## הערות:
- הלוגואים צריכים להיות ברזולוציה גבוהה
- רצוי עם רקע שקוף (PNG)
- הלוגואים יוצגו בגודל 60x60 פיקסלים בדף
- במקרה של שגיאה בטעינה, יוצג הלוגו default-cert.png
